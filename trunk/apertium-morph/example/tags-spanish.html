Cómo escribir una petición de búsqueda

Lea acerca de la sintaxis del lenguaje definido por Lucene para especificar búsquedas

Utilice los siguiente prefijos para especificar términos de búsqueda:

  • lem#: para lemas (todos los posibles lemas de una palabra)
  • lemt#: para lemas (sólo aquellos seleccionados por el desambiguador léxico categorial)
  • tags#: para las etiquetas morfológicas (todas las posibles de una palabra)
  • tagst#: para las etiquetas morfológicas (sólo aquellas seleccionados por el desambiguador léxico categorial)

    • Ejemplo de búsqueda: "lemt#deber de tagst#verbo.infinitivo"
